# Welcome

```
gpupdate /force
```

```
ipconfig /all | clip
```

*   **Check Disk Space**

    ```bash
    wmic logicaldisk get size,freespace,caption
    ```
*   **Power Configuration**

    ```bash
    powercfg /a
    ```
*   **System Information**

    ```bash
    systeminfo | clip
    ```
*   **Network Configuration**

    ```bash
    netstat -ano | clip
    ```
*   **Check Windows Version**

    ```bash
    winver
    ```
